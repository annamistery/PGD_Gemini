# 🧠 AI Psychological Analyzer



Профессиональный инструмент на базе **Gemini 2.5 Pro** для глубокого психологического анализа личности.



## 🚀 Возможности

- **AI Анализ**: Обработка данных пользователя и генерация подробного психотипа.

- **Голосовой отчет**: Озвучка результата с помощью технологии Microsoft Edge TTS.

- **Загрузка**: Возможность скачивания отчётов.

- **Облачный деплой**: Полная поддержка работы в Streamlit Cloud.



## 🛠 Технологии

- **Python 3.11+**

- **Streamlit** (UI Framework)

- **Google Gemini API** (LLM)

- **Edge-TTS** (Voice Generation)
